# TimeO

A tiny native macOS menu bar timer that floats the remaining time above every
Space as a transparent, click-through top overlay.

## Run

```sh
swift run TimeO
```

Or build a launchable app bundle:

```sh
chmod +x scripts/build-app.sh
scripts/build-app.sh
open build/TimeO.app
```

Use the menu bar timer icon to set minutes and start or stop the overlay.
